Context.

Over the last summer, I had the opportunity to work with Professor Julia Poepping on the Carnegie Mellon Information Systems Website. This involved collarborating with CMU Qatar students to restructure and implement the new website with an improved design.

Through this project, I got to touch on many aspects of web design, frontend development, and more using Bootstrap, jQuery, JavaScript, and JSON files. I also had the opportunity to implement some backend features on this website, which can be observed on the "Projects Page" which features a filter feature done entirely in JS.
